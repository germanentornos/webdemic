What are the differences between HTML and XHTML?

HTML and XHTML are both languages in which web pages are written. XHTML was derived from HTML to conform to XML standards. Hence, XHTML is stricter when compared to HTML. The reason for XHTML to be developed was that pages coded in HTML appeared different depending on the browser due to lapses in coding or bad markup.

These are some of the most important differences between them:

Valid in HTML Changes in XHTML to be valid
The DOCTYPE declaration is optional. The DOCTYPE declaration is mandatory.
Example:
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
Some elements can be improperly nested.
Example:
<b><i>Something here</b></i>		 All elements must be properly nested.
Example:
<b><i>Something here</i></b>
Some elements may not be closed.
Example:
<p>This is a paragraph		 All elements must always be closed.
Example:
<p>This is a paragraph</p>
Empty elements may not be closed.
Example:
A break: <br>
An image: <img src="dog.gif" alt="Dog">		 All empty elements must always be closed. This can be done by including a closing tag or by including a slash before the end of the tag.
Example:
A break: <br><br/>
A break: <br/>
An image: <img src="dog.gif" alt="Dog"/>
Elements can be in upper case.
Example:
<BODY><P>This is a paragraph.</P></BODY>		 All elements must be in lower case.
Example:
<body><p>This is a paragraph.</p></body>
Attribute names can be in upper case.
Example:
<table WIDTH="100%">		 Attribute names must be in lower case.
Example:
<table width="100%">
Attribute values may not be enclosed in quotes.
Example:
<table width=100%>		 Attribute values must be enclosed in quotes.
Example:
<table width="100%">
Attribute minimization is allowed.
Example:
<input type="checkbox" name="vehicle" value="car" checked />
<input type="text" name="lastname" disabled />		 Attribute minimization is forbidden.
Example:
<input type="checkbox" name="vehicle" value="car" checked="checked" />
<input type="text" name="lastname" disabled="disabled" />
The first line may not be the root element. The first line has to be the root element <html>, and there can only be one.

The steps to convert from HTML to XHTML are as follow:

  1. Add an XHTML <!DOCTYPE> to the first line of every page.
  2. Add an xmlns attribute to the html element of every page.
  3. Change all element names to lowercase.
  4. Close all empty elements.
  5. Change all attribute names to lowercase.
  6. Quote all attribute values.
